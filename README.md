⚙️ <span style="color:#4CAF50">Setup Instructions</span>
✅ Clone Repository
git clone https://github.com/programer12345anwar/weather-api.git
cd weather-api

✅ Add API Key

Update your application.properties:

weather.api.key=YOUR_WEATHERSTACK_API_KEY
weather.api.url=http://api.weatherstack.com/current

✅ Run Project
mvn spring-boot:run

📡 <span style="color:#FF5722">Example API Endpoint</span>
GET /weather?city=London

✅ <span style="color:#795548">Sample Response</span>
{
  "temperature": 22,
  "humidity": 78,
  "weather": "Cloudy",
  "location": "London"
}
